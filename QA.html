Important FAQ?

What is the DOM in JavaScript?

Answer: The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, with each object corresponding to an element or node in the HTML. Using the DOM, JavaScript can manipulate the content, structure, and style of a web page dynamically.

How do you select an element by its ID in the DOM?

Answer: You can select an element by its ID using the document.getElementById() method.

How can you add a new element to the DOM?

Answer: To add a new element to the DOM, you use document.createElement() to create the element and appendChild() to insert it into the DOM.

How do you change the content of an HTML element using the DOM?

Answer: You can change the content of an HTML element using the innerHTML, textContent, or innerText properties.

How can you remove an element from the DOM?

Answer: You can remove an element using the removeChild() method or the remove() method.

How do you attach an event listener to an element in the DOM?

Answer: You can attach an event listener using the addEventListener() method. This method takes two arguments: the event type (e.g., "click") and the function to run when the event occurs.
