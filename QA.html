What is the difference between let, const, and var?

Answer: var: Function-scoped, can be redeclared, and hoisted. let: Block-scoped, cannot be redeclared in the same scope, hoisted but not initialized. const: Block-scoped, cannot be reassigned after its declaration, hoisted but not initialized.

What are arrow functions in JavaScript?

Answer: Arrow functions are a shorter syntax for writing functions. They do not have their own this, arguments, super, or new.target and are best suited for non-method functions.

What is the difference between == and ===?

Answer: ==: Performs type coercion before comparison (loose equality).
===: Compares both value and type without coercion (strict equality).

What is the this keyword in JavaScript?

Answer: this refers to the object it belongs to. It can change depending on the context: Inside a method: Refers to the object calling the method. Inside a function: Refers to the global object (in non-strict mode). Inside an arrow function: Inherits this from the surrounding lexical context.

What are Promises in JavaScript?

What is event delegation in JavaScript?

Answer: Event delegation is a technique where a single event listener is added to a parent element to handle events of its children, taking advantage of event bubbling.
